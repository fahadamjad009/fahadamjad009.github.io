## ✨ Features

- **Dual-view toggle** — Recruiter view (business impact) and Engineer view (technical depth), filtering the same 15 projects by audience
- **Typewriter role animation** — cycles through Analytics Engineer, Data Scientist, AI Engineer, RegTech Specialist, ML Engineer
- **Animated project cards** — scroll-triggered reveal, mint hover glow, staggered entrance
- **Animated stat counters** — portfolio systems, degrees, GitHub repos, deployed apps
- **Tech stack section** — interactive hover highlighting
- **Space Black aesthetic** — `#0a0a0a` background, `#64FFDA` mint accent, JetBrains Mono + Inter typography
- **Sticky nav** — frosted glass on scroll with mint border reveal
- **Fully responsive** — fluid clamp-based typography and grid layout

## 🚀 Run Locally

```bash
git clone https://github.com/fahadamjad009/fahadamjad009.github.io.git
cd fahadamjad009.github.io
npm install
npm run dev
```

Open `http://localhost:5173`

---

## 📦 Build & Deploy

```bash
npm run build        # outputs to dist/
```

Deployment is automatic — any push to `main` triggers the GitHub Pages workflow and the live site updates within 2–3 minutes.
